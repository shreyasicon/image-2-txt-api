API Reference – CRUD & result format

All endpoints return JSON.

Lambda API base URL — use this for POST, GET, PUT, DELETE
https://xkdvpogqt0.execute-api.us-east-1.amazonaws.com/prod

POST /ocr

Upload an image (multipart form field image) to extract text. Optional body field: language (default: eng). Returns a unique jobId (UUID) — save it to fetch, update, or delete the job later.

Response (200)
{ "success": true, "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890", "filename": "receipt.jpg", "language": "eng", "text": "Extracted text here...", "confidence": 97 }

GET /ocr/:jobId

Fetch a stored OCR result by job ID (the UUID returned from POST /ocr).

Response (200)
{ "jobId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890", "filename": "receipt.jpg", "text": "Extracted text here...", "confidence": 97, "createdAt": "2025-02-16T12:00:00.000Z" }
Error: 404 Job not found
{"error": "Job not found"}

PUT /ocr/:jobId

Update the extracted text for a job. Body: { "text": "New text content" } (JSON, Content-Type: application/json).

Response (200)
{"success": true}
Error: 400 Missing text
{"error": "No text provided"}

DELETE /ocr/:jobId

Delete a stored OCR job by ID.

Response (200)
{"success": true}
